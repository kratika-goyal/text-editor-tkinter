# 📝 Text Editor (Tkinter)

A simple text editor built using Python and Tkinter with features like file handling, themes, and font customization.

![Dark Mode](dark.png)

### Custom Theme

## ✨ Features
- New, Open, Save files
- Save confirmation popup
- Dark / Light mode toggle
- Background & text color customization
- Font size and font family change

##  Tech Used
- Python
- Tkinter

##  How to Run
1. Download the file
2. Run:
   python text_editor.py
